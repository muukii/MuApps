
# Thoughts (Journal App)

## Concept

* その時の気分や気持ちや考えをサッと記録する
* その表現方法はテキストだけとは限らないので、さまざまな表現手法を提供する
  * テキスト
  * 音
  * 写真
  * 手書き
  * Bauhaus
* 一つのポストイットにつき一つのコンテンツというコンセプト
  * それらは繋ぐことができる。Threadのように

## Feature

* 振り返り
  * ポストイット形式なので、画像にできる。
  * それをPhotosに保存することでPhotosが振り返りに使ってくれる
- Sharing
  - デザイン的には洗練されたものを採用してInstagram Storiesにシェアできるようなものにする
    - 北欧のテイストを基本の軸とする
